محرك بحث ذكي لمتاجر ووكومرس يتيح الفلترة المتسلسلة (Dependent Dropdowns) والتوجيه الذكي للمنتجات.

== الوصف ==

**Woo Step-By-Step Finder** هو محرك بحث متطور مصمم خصيصاً للمتاجر التي تحتوي على منتجات ذات سمات معقدة (مثل الإطارات، قطع الغيار).

**المميزات الرئيسية:**
* **بحث متسلسل (AJAX):** القوائم تفتح بناءً على الاختيار السابق.
* **توجيه ذكي:** يوجه العميل للمنتج مباشرة إذا وجد تطابقاً تاماً.
* **تحكم كامل:** لوحة تحكم سهلة لترتيب الحقول وتحديد السمات الرئيسية.
* **تعريب كامل:** واجهة عربية (مصرية) سهلة وودودة.

== التثبيت ==

1. ارفع مجلد `woo-step-finder` إلى مسار `/wp-content/plugins/`.
2. فعل الإضافة من لوحة التحكم.
3. اضبط الإعدادات من قائمة "المحرك الذكي".
4. استخدم الشورت كود `[wsf_search]`.
